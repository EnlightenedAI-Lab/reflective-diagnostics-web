Module 1

Ground Truth Battery

When different AIs give different answers, the first place to test them is where the answer is already known.

This module measures:

  • Accuracy % — how often the model gets the known answer right
  • Avg Confidence — how sure the model is when answering
  • Overconfidence Score — whether confidence stays high when accuracy falls
  • Confident Mistakes — cases where the model is wrong but still highly confident

What we are testing

Externally verifiable domains

  • Mathematics — arithmetic, algebra, derivatives, probability, number theory, combinatorics
  • Bioinformatics — DNA-to-amino-acid translation
  • Cryptography — hashing and exact transformations
  • Graph Theory — shortest paths and relational structure

Ground Truth Battery — takeaway

Known truth turns reliability into a visible question.

What happened

Models can be tested on tasks where the answer is already known or can be verified externally.

What it means

The evaluation is not limited to surface quality. It becomes possible to compare correctness and confidence against truth.

Why it matters

If a model is confidently wrong on known truths, its confidence becomes even less trustworthy where truth is harder to verify.

Module 2

Probe

Live multi-model testing.

  • Parallel — Several models
  • Controlled — Same prompt
  • Evidence — Assigned run ID
  • Export — JSON record

Why it matters

One prompt can produce very different model behavior.

Probe makes those differences visible in a live run before they disappear behind a final answer.

Module 3

Reasoning

Analyzes the structure of a model’s reasoning.

  • Integrity — How well the reasoning holds together
  • Coverage — How much of the answer is supported
  • Orphans — Claims with no support
  • Stability — Consistency across reruns
  • Density — Amount of structural linkage

Module 4

Reflection

Evaluates what changed after reflection.

  • Anchors — Evidence or constraints retained or added
  • Expansion — How much the answer grew
  • Similarity — How much the rewrite changed
  • Inflation — Expansion without support

Module 5

Elasticity

Pressure-tests whether a model keeps its boundary across several turns.

  • Result (Posture) — Did it hold or fail
  • Resistance (MRI) — How strongly it resisted pressure
  • Break Turn — When it first gave way
  • Recovery — Whether it regained the boundary
